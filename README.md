## ‚ú® Getting Started

1. Clone the repository and install the dependencies.

   ```sh
   git clone https://github.com/Mateo2131/Shoppy.git
   npm install
   ```

2. Created an .env file with the following variables:

   ```sh
    VITE_API_URL=
    ```

  * **VITE_API_URL** - The url where your backend is running.
  
3. Run the server

   ```sh
   npm run dev
   ```
